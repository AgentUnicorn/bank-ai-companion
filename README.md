# BANK AI Companion

This repository contains the source code for the BANK AI Companion, a sophisticated financial assistant powered by Google's Gemini API.

View your app in AI Studio: https://ai.studio/apps/drive/1YeOtjnTEBkQT9q-HicNj3UTtaZ3iCM3W

## Run Locally

**Prerequisites:**  Node.js

1.  **Install dependencies:**
    `npm install`
2.  **Set Environment Variable:**
    Create a `.env` file in the root directory and add your Gemini API key:
    `GEMINI_API_KEY=YOUR_API_KEY_HERE`
3.  **Run the app:**
    `npm run dev`
